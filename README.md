# riddance

A tool designed to remove bloatware and restore freedom

## Prerequisites

<details>
<summary>Fedora Linux</summary>

### Python

```
sudo dnf install python3
```

### pip

```
sudo python3 -m ensurepip --upgrade
```
</details>

## Installation

### Linux

```
sudo python3 -m pip install --user riddance
```
